# 🏏 IPL Win Predictor

A Machine Learning powered web application built using **Streamlit** that predicts the winning probability of an IPL match in real-time based on match conditions.

---

## 🚀 Features

* Predicts win probability for both teams
* Interactive UI using Streamlit
* Real-time inputs (score, overs, wickets, target)
* Clean and user-friendly interface
* Supports multiple IPL teams and cities

---

## 🧠 Tech Stack

* **Python**
* **Streamlit**
* **Pandas**
* **Scikit-learn**
* **Pickle** (for loading trained model)

---


## 📊 How It Works

The model takes the following inputs:

* Batting Team
* Bowling Team
* Host City
* Target Score
* Current Score
* Overs Completed
* Wickets Fallen

### 🔢 It calculates:

* Runs Left
* Balls Left
* Current Run Rate (CRR)
* Required Run Rate (RRR)

Based on these features, the model predicts:

* Winning probability of batting team
* Winning probability of bowling team
